pasterd INSTALL
===============

Installation instructions.

Requirements
------------

- [kcgi][], minimal CGI/FastCGI library for C,
- [sqlite][], most used database in the world.

Basic installation
------------------

Quick install.

	$ tar xvzf paster-x.y.z-tar.xz
	$ cd paster-x.y.z
	$ make
	# sudo make install

[kcgi]: https://kristaps.bsd.lv/kcgi
[sqlite]: https://www.sqlite.org
